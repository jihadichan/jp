## Notes

```
NAV 	 = noun, adjective or verb
AN		 = adjective (i = full, na = stem) or noun
Noun	 = noun
A-stem = adj-i without い (e.g. 高), adj-na 
A-i 	 = i-adjective full, not く
A-na 	 = na-adjective stem, no な
AdNo	 = na-adjective or noun 
Sinf	 = sentence informal, e.g. 手紙を書く (object+verb)
Neg		 = negated NAV
(*)		 = optional character, e.g. (か) means that it can be left out
Number+Counter = e.g. 四人, 二台, etc
end		 = sentence end

Vて		= verb て-form
Vます	   = verb masu stem
Vneg	 = verb negative
Vcond	 = verb conditional 
Vnoun	 = noun form of a する verb
```
