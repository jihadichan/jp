# 目

<table><tr>   <td>Summary</td>   <td>A suffix which represents an ordinal number.</td></tr><tr>   <td>English</td>   <td>-th; -th one</td></tr><tr>   <td>Part of speech</td>   <td>Suffix</td></tr></table>
